chip-tools
==

NextThingCo C.H.I.P. Yocto tools.

This layer contains tools to flash the NextThingCo C.H.I.P. boards.

Instructions to use them are available in my Github at https://github.com/myfreescalewebpage/meta-chip.


Contributing
